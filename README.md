# Boilerplate with Create-React-App && Redux

**Prerequisite(s)**

Node >= 8.16.0  
Yarn (or) NPM

## Development Commands

**Yarn**

```
yarn install
yarn start
```

**npm**

```
npm install
npm start
```

## Get the production build

**Yarn**

```
yarn build
```

**npm**

```
npm build
```

## Run the production server

**Yarn**

```
yarn run start:prod
```

**npm**

```
npm run start:prod
```
